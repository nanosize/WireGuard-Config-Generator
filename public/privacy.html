Privacy Policy

1. Information Collection

Server-side

  • nginx access logs:
    • IP address
    • Access timestamp
    • User-Agent string
    • Referrer
  • Umami analytics:
    • Anonymous page-view data
    • Minimal cookies only for session tracking (no personal data)

Client-side

All WireGuard configuration files and QR codes you generate remain entirely in your browser. No configuration data or files are ever sent to our servers or any third party.

2. Purpose of Use

Server-side

  • Site maintenance and operational monitoring
  • Security and incident investigation
  • Anonymous usage statistics (Umami)

Client-side

Generation of WireGuard configs and QR codes within your browser only.

3. Processing Flow

Server-side

  1. nginx receives HTTP request.
  2. Access log entry is recorded.
  3. Page-view event is sent to Umami.
  4. Static files (including this page) are served.

Client-side

  1. Browser loads the page.
  2. CDN-hosted scripts are fetched (e.g. libsodium-wrappers, QR code library).
  3. WireGuard config file is generated locally.
  4. QR code is rendered on the fly.
  5. No generated data is transmitted off your device.

4. Third-Party Resources

  • CDNs (jsDelivr, etc.) for library delivery.
  • Self-hosted Umami instance for anonymous analytics only.
  • No other third-party data sharing or tracking.

5. Security Measures

  • HTTPS enforced (Let’s Encrypt certificate).
  • Static-only site; no sensitive data processed or stored.
  • Risk of data leakage is effectively zero.

6. Policy Updates

This Privacy Policy may be updated at any time. Please refer to this page for the latest version.
