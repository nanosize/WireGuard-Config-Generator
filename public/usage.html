How to Use

This tool helps you generate WireGuard server and client configuration files quickly and easily. QR codes are generated for each client.

  • Number of Clients: Specify how many client peers you need.
  • Network CIDR: Enter the CIDR block for your VPN network (e.g., 10.0.0.0/24).
  • Listen Port: The UDP port the server will listen on (default: 51820).
  • Server Public Endpoint: Your server's public IP or domain with port.
  • DNS: Optional DNS server for clients.
  • Preshared Key: A random key is generated and you can modify it.

After clicking Generate, you can download individual config files, view QR codes, or download all files as a ZIP.
